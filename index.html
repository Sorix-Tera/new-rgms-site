How the site works

Post your screenshot(s) as usual on the corresponding thread on Guild Discord with the following message format: Rank | Region | Name (name is used to identify you in the leaderboard so it can be what you want, as long as it stays consistent).
If you want to add any other notes, make sure to do it on the second line. If you want to keep track of multiple accounts, make sure to use a different name for each account. Enjoy!
